Todo Plugin Installation Guide

Quick Start

Add the following script tag to your HTML:

<script src="https://your-cdn-url/todo-plugin.iife.js"></script>

Initialize the plugin:

<script>
  new TodoPlugin();
</script>

Features

  • Encrypted todo storage
  • Password protection
  • Import/Export functionality
  • Link detection
  • Responsive design

Demo

Click the green checkmark button in the top-right corner to start using the todo list.
